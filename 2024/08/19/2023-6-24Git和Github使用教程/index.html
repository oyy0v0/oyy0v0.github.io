Git和Github使用教程

Git是一个运行在电脑上的版本控制软件 ，而Github是基于这个版本控制软件打造的网站。

Git的三个概念：提交commit，仓库repository,分支branch.

提交 commit

Git可以将你以提交为单位记录下来，当你代码写崩的时候，可以回滚到你最近提交的时候，当你觉得代码没问题了就可以提交了。

仓库 repository

代码库存放在一个文件夹里，而你的每次提交其实就是对应在GIT里面的仓库里面的文件夹的更新，GIT就会记录下来。

分支 branch

可以根据不同的人在不同的分支里编写代码，最后在合起来。

GITHUB

github就可以将你的仓库存放在网站上让大家一起协作开发，大家可以将代码提交更新，在分支上，存放在仓库里，这样就可以一起协作了。
